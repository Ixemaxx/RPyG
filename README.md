Commencé vers le 25 janvier 2026

Présentation:
RPyG c'est un petit projet perso que j'ai créé en janvier 2026 parce que je voulais faire un mini jeu inspiré de Pokémon en python.
Pour faire simple c'est une démo technique qui imite Pokémon dans les graphismes, la manière de se battre, de capture les Pokémons, etc.
On peut comparer ça avec une boite à outils: Avec ça on peut créer pleins de créatures, de dresseurs, d'attaques, de skins, de lieux, dialogues etc. avec un peu de python, 
mais la démo montre quelques possibilités. Bon jeu !

Tuto pour jouer:
- Installer un IDE (genre visual studio)
- Ajouter le dossier du projet dans l'éditeur
- Installer Python 3.12 (c'est pas al seule version, mais elle marche)
- Installer pygame (pip install pygame)
- Lancer main.py

  Touches:
  Z,Q,S,D: Se déplacer
  A: Activer/Désactiver le vélo
  E: Interagir (npc, dialogues etc.)
  X: Ouvrir le menu (ça le fermera pas)
